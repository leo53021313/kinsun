# 主動關懷 v1（定時問候 + 失聯關心）實作計畫

> **For agentic workers:** REQUIRED SUB-SKILL: 用 superpowers:executing-plans 逐任務實作。步驟用核取方塊（`- [ ]`）追蹤。

**Goal:** 排程器定時主動推播：每日對每位長者個人化問候；長者太久沒互動則主動關心。

**Architecture:** `CareAgent.proactive()`（帶記憶用 LLM 組訊息、寫回短期記憶）+ `LineMessenger.push_text()`（LINE push）+ 兩個排程 job（問候／失聯），插進現有 `Scheduler`。`MemoryStore.last_active()` 判失聯。

**Tech Stack:** Python 3.12、google-genai（重用既有）、line-bot-sdk v3、pytest。**無新依賴**。

## Global Constraints

- 採 src layout（`src/kinsun/`），pytest `pythonpath=["src"]`。
- 錯誤隔離：單一長者問候／關心失敗 → 記錄並續跑。
- 失聯用「最後 **user** 發話」+ 每日 job 限流（一天最多一次）。
- 重用 `Scheduler`、`MemoryContext`、`CareAgent`、`run_consolidation`。
- ruff `E,W,F,I,B,UP`、line-length 100。commit 遵循 CONTRIBUTING.md，結尾加 `Co-Authored-By: Claude Opus 4.8 <noreply@anthropic.com>`。

```python
"""主動關懷的排程 job。"""

from __future__ import annotations

import logging
from collections.abc import Callable
from datetime import datetime

from kinsun.scheduler.scheduler import Job

logger = logging.getLogger("kinsun.proactive")

GREETING_INTENT = "早安問候，關心長者今天的狀況"
INACTIVITY_INTENT = "長者已經一段時間沒有互動了，主動表達想念與關心"


def build_greeting_job(
    *,
    sessions: Callable[[], list[str]],
    greet_one: Callable[[str], object],
    hour: int,
    minute: int = 0,
    name: str = "daily-greeting",
) -> Job:
    def run() -> None:
        for session_id in sessions():
            try:
                greet_one(session_id)
            except Exception:  # noqa: BLE001 - 單一長者失敗不影響其他
                logger.exception("問候 session 失敗：%s", session_id)

    return Job(name=name, hour=hour, minute=minute, run=run)


def build_inactivity_job(
    *,
    sessions: Callable[[], list[str]],
    last_active: Callable[[str], float | None],
    clock: Callable[[], datetime],
    threshold_seconds: float,
    care_one: Callable[[str], object],
    hour: int,
    minute: int = 0,
    name: str = "inactivity-care",
) -> Job:
    def run() -> None:
        now_ts = clock().timestamp()
        for session_id in sessions():
            try:
                last = last_active(session_id)
                if last is not None and now_ts - last >= threshold_seconds:
                    care_one(session_id)
            except Exception:  # noqa: BLE001 - 單一長者失敗不影響其他
                logger.exception("失聯關心 session 失敗：%s", session_id)

    return Job(name=name, hour=hour, minute=minute, run=run)
```
