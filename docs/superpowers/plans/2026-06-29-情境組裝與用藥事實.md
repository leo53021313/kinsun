# MemoryContext 情境組裝 + 用藥事實 實作計畫

> **For agentic workers:** REQUIRED SUB-SKILL: 用 superpowers:subagent-driven-development（建議）或 superpowers:executing-plans 逐任務實作。步驟用核取方塊（`- [ ]`）追蹤。

**Goal:** 把一行純轉接的 `MemoryContext` 深化成「注入情境組裝」：每輪把長期記憶 ＋ 長輩當前用藥清單組成一段注入文字；用藥解析內聚在新的 `MedicationFacts` 提供者。

**Architecture:** `MemoryContext` 改為持有 `LongTermStore` ＋ 一個 fact 提供者清單（`FactProvider` Protocol，`facts(session_id) -> str`），`recall` 串接長期記憶與各提供者輸出。新增 `MedicationFacts`：用 `AccountService.elder_by_line` 把 `session_id`（line_user_id）解析成 elder，再 `list_for_elder` 取藥、格式化。用藥每輪固定帶（比照長期記憶的 `HEALTH_QUERY`）。未來「其他事實」加成同型 sibling 提供者。

**Tech Stack:** Python 3.12、pytest。**無新依賴。**

"""用藥事實提供者：把長輩當前用藥組成注入情境的一段文字。"""

from __future__ import annotations

from kinsun.medication.models import slots_label

_PREFIX = "\n這位長者目前固定服用的藥（系統設定的提醒時段，僅供參考、非醫療指示）：\n"


class MedicationFacts:
    """facts(session_id) -> 注入文字。session_id 即長輩的 LINE user_id。"""

    def __init__(self, accounts, medications) -> None:
        self._accounts = accounts
        self._medications = medications

    def facts(self, session_id: str) -> str:
        elder = self._accounts.elder_by_line(session_id)
        if elder is None:
            return ""
        meds = self._medications.list_for_elder(elder.elder_id)
        if not meds:
            return ""
        lines = "\n".join(f"- {m.name}（{slots_label(m.slots)}）" for m in meds)
        return _PREFIX + lines + "\n"
```

"""記憶情境聚合器：長期記憶 ＋ 各事實提供者 → system prompt 注入字串。"""

from __future__ import annotations

from typing import Protocol

from kinsun.longterm.store import LongTermStore


class FactProvider(Protocol):
    def facts(self, session_id: str) -> str: ...


class MemoryContext:
    def __init__(
        self, long_term: LongTermStore, *, facts: list[FactProvider] | None = None
    ) -> None:
        self._long_term = long_term
        self._facts = facts or []

    def recall(self, session_id: str, query: str) -> str:
        out = self._long_term.search(session_id, query)
        for provider in self._facts:
            out += provider.facts(session_id)
        return out
```
