# Mem0 新演算法對齊與清除死 Neo4j 設定 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 移除 mem0 2.0.10 不支援的死 Neo4j 設定/依賴/env，並顯式釘 `version="v1.1"` 對齊新演算法。

**Architecture:** `build_mem0_config` 移除被靜默忽略的 `graph_store` 區塊、加 `version="v1.1"`；`Settings` 移除 `neo4j_*` 三欄位與 `_require`（啟動不再卡 Neo4j 金鑰）；移除 `neo4j` 依賴；更新 env 範本與 living docs。純清理，不改記憶讀寫控制流。

**Tech Stack:** Python 3.12、mem0ai 2.0.10、uv、pytest。
