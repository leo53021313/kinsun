# 危急真實家屬通知 v1 實作計畫

> **For agentic workers:** REQUIRED SUB-SKILL: 用 superpowers:executing-plans 逐任務實作。步驟用核取方塊（`- [ ]`）追蹤。

**Goal:** 把 `LogNotifier`（placeholder）換成 `LineGuardianNotifier`：危急（`tier ≥ L2`）時查綁定資料，依 escalation 順序逐一 LINE push 給所有家屬。

**Architecture:** 儲存層補兩個唯讀查詢（`get_elder_by_line`／`get_guardian`）；`AccountService.guardian_line_ids` 把「長者 LINE id → 各家屬 LINE id」組好；`LineGuardianNotifier` 只依賴兩個小 Protocol（`GuardianDirectory`／`Pusher`），與 `accounts`／`channels` 解耦；`app.py` 注入 `AccountService` + `LineApiMessenger`。

**Tech Stack:** Python 3.12、`psycopg`（v3，已有）、`line-bot-sdk` v3（已有）、pytest。**無新依賴、無新環境變數、無新資料表。**

## Global Constraints

- 採 src layout（`src/kinsun/`），pytest `pythonpath=["src"]`。
- 單元測試全離線：注入 `FakeAccountRepository` 與 spy／fake；Postgres 查詢以 opt-in 整合測試（`KINSUN_IT=1`）覆蓋。
- fail-safe：未綁定／無家屬／push 失敗／查詢例外，`notify` 一律只記錄、**不上拋**、不中斷對話。

    def guardian_line_ids(self, elder_line_id: str) -> list[str]:
        elder = self._repo.get_elder_by_line(elder_line_id)
        if elder is None:
            return []
        line_ids: list[str] = []
        for eg in self._repo.list_elder_guardians(elder.elder_id):

```python
class GuardianDirectory(Protocol):
    def guardian_line_ids(self, elder_line_id: str) -> list[str]: ...


class Pusher(Protocol):
    def push_text(self, user_id: str, text: str) -> None: ...


_ALERT_PREFIX = "⚠️【金孫關懷提醒】"


def _format_alert(assessment: RiskAssessment) -> str:
    text = (
        f"{_ALERT_PREFIX}您關心的長輩可能需要您的注意："
        f"{assessment.reason}（風險等級 {assessment.tier.name}）。請盡快主動關心一下。"
    )
    if assessment.tier == RiskTier.L3:
        text += "（如情況緊急，請自行評估是否撥打 119。金孫不提供醫療診斷。）"
    return text


class LineGuardianNotifier:
    """危急時依升級順序推播給所有家屬。任何失敗只記錄、不中斷對話。"""

    def __init__(self, directory: GuardianDirectory, pusher: Pusher) -> None:
        self._directory = directory
        self._pusher = pusher

    def notify(self, session_id: str, assessment: RiskAssessment) -> None:
        try:
            targets = self._directory.guardian_line_ids(session_id)
            if not targets:
                logger.warning(
                    "危急但查無可通知家屬 session=%s tier=%s reason=%s",
                    session_id,
                    assessment.tier.name,
                    assessment.reason,
                )
                return
            text = _format_alert(assessment)
            sent = 0
            for line_id in targets:
                try:
                    self._pusher.push_text(line_id, text)
                    sent += 1
                except Exception:  # noqa: BLE001
                    logger.exception("推播家屬失敗 line_id=%s", line_id)
            logger.warning(
                "已通知家屬 session=%s tier=%s 成功=%d/%d",
                session_id,
                assessment.tier.name,
                sent,
                len(targets),
            )
        except Exception:  # noqa: BLE001
            logger.exception("家屬通知流程異常 session=%s", session_id)
```
